# Title: 기술적 지표 기반 트레이딩 전략 구현 및 MDD 최적화 백테스팅

Description: 볼린저 밴드, 이동평균선, Trailing Stop Loss 로직이 포함된 파이썬 백테스팅 스크립트를 작성하고, 과거 데이터를 기반으로 수익률과 MDD를 분석하여 파라미터를 자율 튜닝하는 워크플로우입니다.

## Steps
1. 사용자가 요청한 종목(티커)과 타임프레임, 백테스팅 기간을 확인하세요.
2. `yfinance` 또는 로컬 CSV 데이터를 로드하여 OHLVC(시가/고가/저가/종가/거래량) 데이터프레임을 생성하는 파이썬 스크립트를 작성하세요.
3. 다음 지표를 계산하는 로직을 코드에 포함하세요:
   - 이동평균선 (5일, 20일, 60일, 120일)
   - 볼린저 밴드 (상단, 하단, 표준편차 2)
   - Trailing Stop Loss (진입 최고가 대비 사용자 지정 % 하락 시 손절)
4. 진입/청산 시그널과 함께 누적 수익률, 승률, MDD(Maximum Drawdown)를 산출하는 백테스팅 루프를 실행하세요.
5. [Self-Repair Loop] 초기 백테스팅 결과의 MDD가 과도하게 높거나(예: -15% 이상) 수익률이 저조할 경우, Trailing Stop 비율이나 이동평균선 정배열 조건을 스스로 보정하여 재실행하세요.
6. 최적의 파라미터 조합, 거래 일지(Trade Log), 그리고 최종 누적 수익률 그래프를 아티팩트(Artifact) 뷰어로 시각화하여 리포트하세요.
