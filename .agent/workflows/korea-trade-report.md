# 6월 한국 수출입 통계 분석 리포트 생성 워크플로우

## Global Context
| Variable | Description | Example |
|----------|-------------|---------|
| `{{TARGET_MONTH}}` | 분석 대상 월 | `2026년 6월` |
| `{{OUTPUT_PATH}}` | 리포트 저장 경로 | `reports/trade/trade_report_2026_06.md` |

---

## Workflow Phases

### Phase 1: 데이터 수집 및 검증 (Data Retrieval)
* **Instructions:**
  1. `/browser` 명령어를 실행하여 관세청(Korea Customs Service) 또는 산업통상자원부의 최신 '2026년 6월 수출입 동향' 보도자료 및 통계 데이터를 검색하세요.
  2. 주요 지표인 **수출액, 수입액, 무역수지(Trade Balance)** 총액 데이터를 확보해야 합니다.
  3. 반도체, 자동차, 석유화학 등 **주요 품목별 수출 실적**과 미국, 중국, EU 등 **주요 국가별 실적** 데이터를 함께 수집하세요.
  4. 수집된 원본 데이터의 신뢰성을 검증하고, 누락된 지표가 있다면 보완 검색을 수행하세요.

### Phase 2: 데이터 구조화 및 통계 분석 (Data Analysis)
* **Instructions:**
  1. 수집한 수치 데이터를 바탕으로 전년 동월 대비(YoY) 증감률을 계산하세요.
  2. 무역수지가 흑자 또는 적자인지 판별하고, 기저 효과나 특정 품목의 견인 효과가 있는지 분석하세요.
  3. `{{OUTPUT_PATH}}`에 작성할 마크다운 테이블 구조를 먼저 설계하세요. (품목별/국가별 테이블 필수 포함)

### Phase 3: 리포트 초안 작성 (Report Generation)
* **Instructions:**
  1. 분석된 내용을 바탕으로 전문적이고 가독성이 높은 마크다운 리포트를 작성하세요.
  2. 리포트는 반드시 다음 구조를 따라야 합니다:
     - **요약문 (Executive Summary):** 3줄 요약 및 주요 시사점
     - **종합 수출입 동향:** 총 수출/수입액 및 무역수지 현황 (YoY 비교)
     - **품목별/지역별 상세 분석:** 표(Table)와 텍스트를 조합한 세부 지표 분석
     - **향후 전망 및 리스크 요인:** 글로벌 공급망 및 환율 변동 등 고려
  3. 작성된 리포트는 `{{OUTPUT_PATH}}` 파일로 저장하세요.

### Phase 4: 최종 검증 및 포맷팅 (Review & Verification)
* **Instructions:**
  1. 작성된 리포트 파일(`{{OUTPUT_PATH}}`)을 읽어와서 숫자가 올바르게 매칭되었는지, 오탈자가 없는지 최종 검토하세요.
  2. 마크다운 문법(하이라이트, 인용구 등)이 가독성 있게 적용되었는지 검증하세요.
  3. 검증이 완료되면 "6월 한국 수출입 통계 분석 리포트 생성이 완료되었습니다." 메시지와 함께 결과 파일 경로를 사용자에게 제안하세요.
