# Title: 대형 기관 13F 공시 분석 및 포트폴리오 매매 동향 추적

Description: 워런 버핏 등 주요 투자 대가의 최신 13F Filing 데이터를 파싱하여 포트폴리오 내 종목별 비중 변화(신규 매수, 전량 매도, 비중 조절)를 일목요연하게 분석하는 워크플로우입니다.

## Steps
1. 분석 대상 기관(예: Berkshire Hathaway)과 대상 분기를 확인하세요.
2. SEC EDGAR API 또는 관련 데이터 소스를 통해 해당 기관의 최신 13F XML/텍스트 데이터를 수집하는 스크립트를 구동하세요.
3. 수집된 데이터를 바탕으로 다음 항목을 추출 및 가공하세요:
   - 직전 분기 대비 신규 편입 종목 (New Positions)
   - 직전 분기 대비 전량 매도 종목 (Sold Out)
   - 보유 비중이 5% 이상 증가 또는 감소한 종목 (Weight Changes)
4. 기관의 상위 10개 보유 종목(Top 10 Holdings)의 비중을 파이 차트 형태로 표현할 수 있도록 데이터를 정렬하세요.
5. 분석된 매매 동향 요약본과 섹터별 비중 변화를 Antigravity 아티팩트(Markdown Table 및 차트 구조)로 깔끔하게 렌더링하여 사용자에게 제시하세요.
